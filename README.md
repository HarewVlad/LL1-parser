# LL1-parser

**Format:**  
NT -> 'type' 'name' ...  
**Ex:**  
"S -> NT F SPECIAL | T epsilon" --- "S -> F | epsilon"    
**How to use: **  
-Enter regular expression("exit" to stop)  
-Enter start nonterminal  
-See all of this stuff that's popped out  
-Enter string to parse("exit" to stop)
